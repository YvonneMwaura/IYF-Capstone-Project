
BizConnect Marketplace
Short Description
BizConnect is a B2B2C marketplace platform that connects factories, businesses, and customers within a unified digital ecosystem. It enables efficient discovery, interaction, and collaboration across different levels of the supply chain.
Purpose of the Project
BizConnect was developed to simplify business operations by creating a centralized platform where manufacturers, businesses, and consumers can connect directly. It addresses the challenge of fragmented supply chains by improving accessibility, visibility, and communication between all parties involved in commerce.
Tech Stack
Frontend
Lovable – Used to design and build the user interface and overall frontend experience.
Backend / Database
Supabase – Intended to handle database management, authentication, and backend services as the project evolves into a full-stack application.
AI Integration
Hugging Face – Intended for integrating AI models to provide intelligent features such as recommendations, insights, and automation.
Deployment
Vercel – Used for hosting and deploying the frontend application.
Key Features
User account creation and authentication (planned)
Marketplace connecting factories, businesses, and customers
Search and discovery of products and services
User interaction and communication features (planned)
AI-powered recommendations and insights
Responsive and user-friendly interface
Installation / Setup Steps
Clone the repository to your local machine.
Navigate into the project directory.
Install the required dependencies using a package manager such as npm.
Required Environment Variables
REACT_APP_SUPABASE_URL – Supabase project URL
REACT_APP_SUPABASE_ANON_KEY – Supabase public API key
REACT_APP_HUGGINGFACE_API_KEY – Hugging Face API key
These variables should be stored in a .env file and must not be committed to version control.
How to Run the Project
Start the development server using your package manager. The application will run locally, typically on a development server environment such as localhost.
