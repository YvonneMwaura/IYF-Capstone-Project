IYF Capstone Project: BizConnect Marketplace
Short Description
A unified digital platform where users can purchase products and services while exploring a comprehensive directory of verified businesses and manufacturing factories.
Purpose of the Project
This project addresses the gap between industrial production and consumer access. It provides a centralized space for factories to gain visibility and for service providers to reach a wider audience, simplifying the discovery-to-purchase pipeline for users.
Tech Stack
Frontend: React (for a dynamic, responsive UI)
Backend/Database: Supabase (handling PostgreSQL database and real-time updates)
AI Integration: Hugging Face (used for smart product recommendations or categorizing listings)
Authentication: Supabase Auth
Key Features
Dual Marketplace: Integrated shopping experience for both physical products and professional services.
B2B/B2C Directory: Categorized listings for factories and local businesses with location-based filtering.
Role-Based Access: Distinct interfaces for buyers, business owners, and factory managers.
Lead Generation: Direct inquiry system for high-volume factory orders.

Use code with caution.

Install dependencies:
bash
cd IYF-Capstone-Project
npm install
Use code with caution.

Set up Environment Variables: Create a .env file in the root directory (see below).
Required Environment Variables
To run this project, you will need to add the following variables to your .env file:
REACT_APP_SUPABASE_URL: Your unique Supabase project URL.
REACT_APP_SUPABASE_ANON_KEY: Your Supabase anonymous API key.
HUGGINGFACE_API_TOKEN: API token for AI-powered features.
How to Run the Project
In the project directory, you can run:
bash
npm start
Use code with caution.

The app will open in your browser at http://localhost:3000.
